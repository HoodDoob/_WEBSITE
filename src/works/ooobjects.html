Ooobjects

Vinhanen, Copenhagen, 2024

O0objects is a project about bending reality using digital tools. Itâ€™s a collection of 3D-printed objects, each with a different set of characteristics, that create a consistent collection challenging human perception and posing questions about the line between the digital and the real world. Each object we created is processed, skewed and distorted, highlighting its natural dimensionality. A golden fruit that bends evolution, an old, vertical TV, a skewed venus - they all question standards of usability, beauty and logic.
